# HashNode Pro - Bulut Madenciliği Simülatörü

Bu proje, kullanıcıların bulut madenciliği dünyasını deneyimlemesi için tasarlanmış bir mobil uygulama simülasyonudur. 

## 🚀 Proje Hakkında
HashNode Pro, bir oyun ve eğitim aracıdır. Cihazın işlemci gücünü kullanmadan (simülasyon yoluyla) madencilik mekaniklerini öğretmeyi hedefler.

## ⚠️ Önemli Uyarı
Bu uygulama gerçek bir kripto para kazımı yapmaz. Tamamen oyun içi ekonomiye dayalı bir simülasyon projesidir.
